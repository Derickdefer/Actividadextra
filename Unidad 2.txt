 1. Conceptos básicos para iniciar a programar
Programación: Crear instrucciones para que una computadora realice tareas.
Lenguaje de programación: Herramienta para escribir código (ej: Python, Java).
Variables: Espacios en memoria para almacenar datos (ej: nombre = "Juan").
Sintaxis: Reglas para escribir código correctamente.
Depuración: Proceso de encontrar y corregir errores en el código.
2. Tipos de datos y operadores
Tipos de datos: Clasificación de la información que maneja un programa:
Enteros (int): Números sin decimales (ej: 5).
Flotantes (float): Números con decimales (ej: 3.14).
Cadenas (str): Texto (ej: "Hola").
Booleanos (bool): Valores True o False.
Operadores:
Aritméticos: +, -, *, / (suma, resta, multiplicación, división).
Comparación: ==, !=, >, < (igual, diferente, mayor, menor).
Lógicos: and, or, not (para combinar condiciones).

3. Estructuras de control condicionales
Permiten tomar decisiones en un programa basadas en condiciones.
If: Ejecuta un bloque de código si se cumple una condición.
Else: Ejecuta un bloque si la condición del if no se cumple
4. Estructuras de control: bucles
Permiten repetir un bloque de código varias veces.
For: Itera sobre una secuencia (lista, rango, etc.).
While: Repite un bloque mientras se cumpla una condición.
Break y continue:
break termina el bucle.
continue salta a la siguiente iteración.
